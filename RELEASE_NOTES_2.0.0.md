# BetterExperience v2.0.0

## 简介

`v2.0.0` 是一次较大规模的重构更新，重点放在配置系统、配置体验、热键系统和项目可维护性上。

相较于 `v1.0.0`，本版本引入了独立配置系统、内置可视化配置界面、中英双语支持、强类型热键与手柄输入支持，并完成了对 `Alice In Cradle ver029i` 的适配。

## 主要更新

### 配置系统重构

- 不再依赖 `BepInEx` 默认配置绑定与 `ConfigurationManager`
- 配置文件调整为：`BepInEx/plugins/BetterExperience/BetterExperience.cfg`
- 配置项现在可保留分组名、条目名和描述信息
- 优化了配置保存与解析逻辑，提升稳定性与可维护性

### 内置配置界面

- 新增游戏内可视化配置界面，默认按 `F1` 打开
- 支持滑块、枚举选择、热键录制、修改提示
- 支持点击窗口外自动隐藏
- 支持 `简体中文` / `English`

### 热键系统升级

- 支持组合键
- 支持多个备选热键
- 支持手柄输入
- 默认热键：
  - `F1`：打开/关闭配置界面
  - `Ctrl+R`：重新加载配置
  - `F`：一键刷新商店
  - `Ctrl+T`：重新加载贴图

### 功能扩展与改进

- 新增危险度设置功能
- 保留并扩展现有的体验优化、数值调整、限制解除、贴图替换等功能
- 改善了配置与 GUI 的底层结构，便于后续继续扩展

### 修复与稳定性

- 修复重新加载贴图时可能发生的内存泄漏问题
- 修复加载存档后 tooltip 样式失效的问题
- 修复 GUI 可能重复初始化的问题
- 修复若干配置层相关问题，包括比较逻辑、隐式转换歧义、表名和描述保存异常等

### 测试补强

- 为配置核心、配置 GUI、热键系统、样式资源和辅助模块补充了较完整的单元测试
- 提升了后续迭代的可验证性

## 兼容版本

- `Alice In Cradle ver029i`
- `BepInEx v5.4.23.5`

## 安装说明

1. 确保已安装 `BepInEx 5`
2. 将 `BetterExperience.dll` 放入 `BepInEx/plugins/BetterExperience/`（或 `BepInEx/plugins/`）
3. 启动游戏一次以生成配置文件与相关目录
4. 如需调整设置，可直接编辑配置文件，或在游戏中按 `F1` 打开内置配置界面

## 升级提示

- 从旧版本升级时，请注意配置文件路径和配置结构已经调整
- 旧版依赖 `ConfigurationManager` 的使用方式已不再适用
- 若你使用了贴图替换功能，请检查贴图目录是否位于 `BepInEx/plugins/BetterExperience/ReplaceTexture/`
